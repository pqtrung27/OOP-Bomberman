### Requirement ###
[JDK8 Environment](https://www.oracle.com/java/technologies/javase/javase8-archive-downloads.html)


### How to start ###
- Go to
```
"\out\artifacts\Bomberman_jar"
```

- Run this in command prompt
```
java -jar bomberman.jar
```
